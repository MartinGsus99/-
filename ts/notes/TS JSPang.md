# TypeScript

## 一、TypeScript

### 1.TS

- JS超集 ：静态类型+面向对象；
- 大型应用 多人协作；

| 类型   | TS                             | JS         |
| ------ | ------------------------------ | ---------- |
| 应用端 | 应用程序                       | 客户端语言 |
| 集合   | JS超集，包括JS，最终编译为JS   |            |
| 平台   | 跨平台、开源                   |            |
| 起始   | 开始于JS、终止于JS             |            |
| 重用   | 可以重用，可以引入JS流行库；   |            |
| 类     | 类、接口、模块；更易于组件性； |            |

### 2.基础操作

```shell
#编译为js
tsc './file.ts';
```
